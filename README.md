<h4>; &lt;Sum&gt;:</h4>
<p style="text-align:justify"><strong>Slippery Clip</strong> is an optimized and portable clipboard management utility, mostly powered by minimalistic GUI with maximum keyboard support in mind. It provides easy-to-use storage mechanism for logging and processing
 any data sent to clipboard, complete with hotkey-based multipasting functionality.</p>
<hr>
<h4>; &lt;Featurae&gt;:</h4>
<p><u><em><strong>--&gt;</strong> Overcoming innate limitations &amp; enchancing general functionality of Windows' imbedded clipbuffer.</em></u><br>
<strong>-&gt;</strong> Rather vast array of hotkeys, rendering ~80% of functionally to be accessible from keyboard.<br>
<strong>-&gt;</strong> XCopy-based installation process. Single ‘SlipperyClip.exe’ is enough for functioning.<br>
<strong>-&gt;</strong> Multitask data viewing system with no limits about simultaneous window opening.<br>
<strong>-&gt;</strong> Unobtrusive PreXP-stylish design with optional opacity and border snapping.<br>
<strong>-&gt;</strong> Raw text, HTML, bitmap, RTF, metaimage and path listings support.<br>
<strong>-&gt;</strong> Up to ten hotslots for rapid multipasting emulation (Ctrl+[0-9]).<br>
<strong>-&gt;</strong> Plain .INI-based settings storage with no access to registry ever.<br>
<strong>-&gt;</strong> Asynchronous search engine with regexp and reparsing support.<br>
<strong>-&gt;</strong> Intercessional data preservation within single dedicated file.<br>
<strong>-&gt;</strong> Consistent runtime autocompression for all absorbed data.<br>
<strong>-&gt;</strong> Basic Drag/Drop support for outside communications.<br>
<strong>-&gt;</strong> RUS&lt;-&gt;ENG rapid xlat conversion (hotkey available).<br>
<strong>-&gt;</strong> Complete Unicode-compilancy.</p>
